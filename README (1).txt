Our program consists of 7 pdf files comprising of code, 2 audio files, and 9 images. It is compiled using javac *.java and ran using run Main. The game has 2 levels. On the very first page, the user will choose the difficulty level and click on play game to start the game. Game is played by moving the saucer to make the ball bounce and break the bricks, so the user will control the movement of saucer. It is moved to the left using the key 'a' and to the right using 'd'. After the first level is completed, the user will be given the option to quit or play the next level.
